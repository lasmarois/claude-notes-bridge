Setup Instructions

Step 1: Grant Full Disk Access

The bridge reads Apple Notes' database directly and needs Full Disk Access:

  1. Open System Settings > Privacy & Security > Full Disk Access
  2. Click the + button
  3. Add your terminal app (Terminal, iTerm2, or Warp) for the CLI tool
  4. Add NotesSearch.app from /Applications for the search app

Step 2: Configure Claude Code (CLI users)

The installer will try to configure this automatically. If it doesn't, run:

claude mcp add apple-notes --scope user -- /usr/local/bin/claude-notes-bridge serve

Or add manually to your project's .mcp.json:

{
  "mcpServers": {
    "apple-notes": {
      "command": "/usr/local/bin/claude-notes-bridge",
      "args": ["serve"]
    }
  }
}

Step 3: Verify

claude-notes-bridge --version
claude-notes-bridge list --limit 3
